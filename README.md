# Event Ride
Ionic app for Destrua.me project

# Instalação

```
npm install -g cordova ionic
git clone https://github.com/Thithi32/eventride_app
ionic serve
```

## Para Android

```
ionic add plateform android
ionic build android
ionic emulate android
```

## Para IOS

```
ionic add plateform ios
ionic build ios
ionic emulate ios
```
